# Backend Stream — Decisions Log

---

## 2026-04-28 — CRUD через Livewire (не через Controller+Blade)
**Решение**: Все списки/формы/детали — Livewire-компоненты.
**Причина**: меньше boilerplate, прямой Eloquent, live-валидация. См. CLAUDE.md.

## 2026-04-28 — Slide-over для форм (не модалки и не отдельные страницы)
**Решение**: Создание/редактирование открывается в slide-over панели.
**Причина**: пользователь не теряет контекст списка.

## 2026-04-28 — App\Helpers\Acl как единая точка проверки прав
**Решение**: Все проверки прав — через `Acl::can(...)` или Blade-директиву `@acl`.
**Причина**: Работает в preview-mode (без Spatie) и в production. Не нужно везде писать `auth()->user()->can()`.

## 2026-04-28 — Service Layer для сложной логики
**Решение**: Сложная бизнес-логика выносится в `app/Services/` (QuoteService, InvoiceService, PdfService).
**Причина**: тонкие модели + тонкие компоненты = тестируемая архитектура. Логика, не привязанная к одной модели, идёт в сервис.
