# Glossary — Общий словарь RSG-CRM

Единые термины, чтобы все потоки говорили на одном языке.

## Бизнес-сущности

| RU | EN (код) | Что это |
|------|---------|---------|
| Лид | `Lead` | Потенциальный клиент, ещё не сделка |
| Клиент | `Customer` | Компания-клиент (ЮЛ) с реквизитами |
| Контактное лицо | `Contact` | Физлицо в компании-клиенте |
| КП | `Quote` (коммерческое предложение) | Предложение с позициями товаров |
| Инвойс | `Invoice` | Счёт на оплату |
| Заказ | `Order` | Заказ из портала клиента |
| Тикет | `Ticket` | Обращение клиента в тех. поддержку |
| Заявка | `EquipmentRequest` | Запрос на оборудование, которого нет в каталоге |
| Активность | `Activity` | Запись в timeline лида/клиента (звонок, письмо, заметка) |

## Каталог

| RU | EN | Описание |
|------|-----|---------|
| Категория | `Category` | Иерархическая (POS / Весы / Сканеры) |
| Товар | `Product` | Карточка товара с SKU |
| SKU / артикул | `sku` | Уникальный артикул для бизнеса |
| Закупочная цена | `cost_price` | Видна только админу/закупщику |
| Розничная цена | `retail_price` (RRP) | Стандартная для клиентов |
| Оптовая цена | `wholesale_price` | От N единиц |
| Спец-цена | `special_price` | Индивидуально для клиента |
| Остаток | `stock` | Количество на складе |

## Процессы

| Термин | Объяснение |
|--------|------------|
| Воронка продаж | Sales funnel: путь Лид → КП → Won/Lost |
| Конверсия | % перехода между этапами воронки |
| SLA | Service Level Agreement — нормативное время реакции на тикет |
| CSAT | Customer Satisfaction — оценка клиента после закрытия тикета |
| Score (лида) | Оценка приоритета 1-10 |
| Sales-pipeline / pipeline | Список всех открытых сделок менеджера |

## Статусы

### Lead
`new` → `qualified` → `contacted` → `in_negotiation` → `won` / `lost`

### Quote (КП)
`draft` → `sent` → `viewed` → `accepted` / `rejected` → `expired`

### Invoice
`draft` → `sent` → `partially_paid` → `paid` → `overdue` → `cancelled`

### Ticket
`open` → `in_progress` → `pending_customer` → `resolved` → `closed`

### Equipment Request
`submitted` → `under_review` → `quoted` → `closed`

## Контуры системы

| Термин | Что это |
|--------|---------|
| **Внутренний контур** / CRM / admin | Для сотрудников RSG. Routes `/admin/*`, layout admin.blade.php |
| **Внешний контур** / Portal | Для клиентов. Routes `/portal/*`, layout portal.blade.php |

## Роли (см. также config/permissions.php)

- `super-admin`, `sales-director`, `sales-manager`, `tech-support`, `catalog-manager`, `accountant` — внутренние
- `client-admin`, `client-user` — клиентские

## Региональные термины

| Термин | Значение |
|--------|----------|
| ИНН | Идентификационный номер налогоплательщика (UZ) |
| ОКЭД | Общий классификатор видов экономической деятельности |
| НДС | Налог на добавленную стоимость, в UZ — 12% |
| UZS | Узбекский сум (основная валюта) |
| USD | Доллар США (используется для импортных товаров) |
