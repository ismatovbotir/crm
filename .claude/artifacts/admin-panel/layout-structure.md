## Размеры

- **Header**: `h-16` (64px), `bg-white`, `border-b border-gray-200`
- **Sidebar**: `w-64` (256px), `bg-white`, `border-r border-gray-200`
- **Content**: `bg-gray-50`, `p-6`, `min-h-screen`

### Состояния пунктов меню
- **Default**: `text-gray-600 hover:bg-gray-50 hover:text-gray-900`
- **Active**: `bg-primary-50 text-primary-700 border-l-2 border-primary-600`
- **Item structure**: иконка (w-5 h-5) + текст + (опц.) badge с числом

## Header (Top bar)

```
┌────────────────────────────────────────────────────────────────────┐
│ ☰   🔍 [Поиск по клиентам, лидам, КП...]    🔔  💬  👤 Алекс ▾   │
└────────────────────────────────────────────────────────────────────┘
```

### Элементы (слева направо)
1. **Гамбургер** (на mobile/tablet) — `< lg` показан, открывает сайдбар
2. **Глобальный поиск** — `w-96` на десктопе, через Livewire (поиск по сущностям)
3. **Notifications** (🔔) — dropdown с последними уведомлениями + красная точка для unread
4. **Quick chat / messages** (💬) — для будущего внутреннего чата (опц.)
5. **User menu** (аватар + имя + ▾) — dropdown с: Профиль / Настройки / Выйти

## Page Header (внутри content)

Стандартизированный блок для каждой страницы:

```
┌────────────────────────────────────────────────────────────┐
│  Лиды / Список                                             │
│  Все потенциальные клиенты      [Filter ▾]  [+ Создать]   │
└────────────────────────────────────────────────────────────┘
```

- **Breadcrumbs**: `text-sm text-gray-500` с разделителем `/`
- **Title**: `text-2xl font-semibold text-gray-900`
- **Description**: `text-sm text-gray-500`
- **Actions** (справа): Filter, Export, Create — кнопки в один ряд

## Mobile / Tablet поведение

- **< 768px**: контент становится одноколоночным, сайдбар скрыт за overlay
- **768-1023px**: сайдбар скрывается, открывается по гамбургеру с overlay
- **>= 1024px**: фиксированный layout с сайдбаром

## Z-index стек

```
50  — Modal / Dialog
40  — Dropdown menu
30  — Mobile sidebar overlay
20  — Sticky header
10  — Sidebar
 0  — Content
```
