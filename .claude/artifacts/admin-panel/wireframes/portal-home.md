## Layout особенности

- **Header**: горизонтальный (не сайдбар, как в админке) — проще для клиента
- **Меню**: Каталог / Заказы / КП / Инвойсы / Тикеты / Профиль
- **Цвет**: тот же primary blue, но фон может быть `bg-gradient-to-br from-primary-50 via-white to-gray-50`
- **Кнопки**: больше padding, более крупный font (для удобства)

## Quick action cards (внизу)

3 крупные карточки для частых действий:
1. **Каталог** → `/portal/catalog`
2. **Заявка на оборудование** → `/portal/equipment-requests/create`
3. **Поддержка** → `/portal/tickets/create`

## Поведение

- При входе клиента: redirect на `/portal` (его дашборд)
- Если у клиента нет активности — показываем onboarding с 3 шагами:
  1. Заполните профиль компании
  2. Просмотрите каталог
  3. Свяжитесь с менеджером

## Адаптив

- На mobile: карточки KPI идут по 2 в ряд (вместо 4)
- Quick action cards — стек по вертикали

## Notifications для клиента

В правом верхнем углу — колокольчик с уведомлениями:
- Получено КП
- Получен инвойс
- Ответ на тикет
- Заказ отгружен
- Заказ доставлен
