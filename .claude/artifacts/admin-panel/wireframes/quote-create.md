URL: `/admin/quotes/create`

## Ключевые механики

### Автокомплит товаров
- Поле поиска: ввод 2+ символов → запрос к каталогу
- Результат — выпадающий список: фото товара, название, SKU, цена для этого клиента
- Клик → товар добавляется в позиции

### Цены
- При выборе товара подтягивается цена для **типа клиента** (опт / спец)
- При смене клиента — пересчёт всех позиций (с предупреждением)
- Скидка на позицию: % или абсолют (toggle)

### Валюты
- При смене валюты UZS↔USD пересчёт по курсу из настроек
- Курс отображается рядом с итогом

### НДС
- Toggle "С НДС / Без НДС" в шапке (на основе типа клиента, можно override)
- Ставка из настроек (по умолчанию 12% для UZ)

### Drag & drop
- Позиции можно переупорядочивать перетаскиванием
- Удаление позиции: ✕ или Delete (с undo через toast)

### Live preview
- Кнопка "Preview PDF" → открывает превью в новой вкладке/модалке
- PDF генерируется на лету, использует фирменный шаблон с логотипом

### Validation
- Минимум 1 позиция
- Клиент обязателен
- Срок действия не в прошлом

## Действия после сохранения

1. **"Save Draft"** → сохраняет статус Draft, остаётся в форме
2. **"Сохранить и отправить"** → сохраняет, генерирует PDF, открывает модалку отправки email
3. **После отправки**: статус Sent, появляется кнопка "Track" в карточке КП
