## Компоненты

1. **4 KPI-карточки** в ряд — `<x-stat-card>`
2. **Linе chart** — продажи по неделям (Chart.js)
3. **Pie chart** — лиды по источникам
4. **Таблица последних лидов** — Livewire компонент с lazy loading

## Поведение

- Период (текущий месяц / квартал / год) переключается в правом верхнем углу контента
- Клик по KPI-карточке → переход в соответствующий модуль с фильтром
- Клик по строке таблицы → открыть карточку лида
