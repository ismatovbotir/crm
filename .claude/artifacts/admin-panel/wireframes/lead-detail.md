## Header (sticky)

- Имя, компания, статус (clickable dropdown для смены)
- Метаданные: менеджер, дата создания
- Action buttons:
  - **Изменить** — open модалку/sheet с формой
  - **Позвонить** — `tel:` link или интеграция с CallCenter
  - **→ КП** — создать КП на этого лида
  - **⚙ ⋯** — dropdown: Конвертировать в клиента / Передать другому / Удалить

## Tabs

1. **Обзор** — основная инфо + сводка + заметки + активность
2. **Активность** — полная лента с фильтрами по типу
3. **Задачи** — список задач + календарь напоминаний
4. **КП** — связанные коммерческие предложения
5. **Документы** — прикреплённые файлы

## Sections в Обзоре

### Контактная информация
2-колоночный layout (на mobile — одна колонка). Поля inline-editable (клик → input).

### Сводка
- Источник, score, бюджет, тип бизнеса, регион
- Можно добавлять кастомные поля через настройки

### Заметки
- Rich text (Markdown lite)
- История изменений (кто, когда)

### Лента активности (Timeline)
- Иконка + время + текст
- Типы: Звонок / Email / КП / Встреча / Заявка с сайта / Заметка / Изменение статуса
- "+ Добавить активность" — dropdown с типами

## Действия со статусом

Клик на бейдж статуса → выпадающий список:
- Quick-switch с подтверждением
- "Перейти в Won" → откроется модалка с финальной суммой и комментарием
- "Перейти в Lost" → модалка с причиной (lost-reason: цена / сроки / конкурент / другое)

## Right side (опционально, на широких экранах)

Можно добавить sidebar справа со связанными сущностями:
- Связанный клиент (если конвертирован)
- Открытые тикеты
- Последние взаимодействия
