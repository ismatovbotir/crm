## Импорт CSV

Кнопка "Импорт" → модалка:

```
┌────────────────────────────────────────────┐
│ Импорт товаров                          ✕  │
├────────────────────────────────────────────┤
│  1. Скачайте шаблон [CSV]                  │
│  2. Заполните данными                      │
│  3. Загрузите файл:                         │
│                                              │
│  [Drop файл сюда или кликните]             │
│                                              │
│  Опции:                                     │
│  ☐ Обновить существующие (по SKU)          │
│  ☑ Создать новые                            │
│  ☐ Удалить отсутствующие в файле            │
│                                              │
│              [Отмена]  [Загрузить]          │
└────────────────────────────────────────────┘
```

После загрузки — превью + подтверждение:

```
Найдено в файле: 87 строк
  ✓ Будет создано: 12
  ↻ Будет обновлено: 73
  ⚠ Ошибок: 2 (см. детали ниже)

[Скачать отчёт] [Отмена] [Применить]
```

## Bulk-действия

Чекбоксы в карточках/списке → массовые действия:
- Опубликовать / Снять с публикации
- Скрыть/показать в портале
- Изменить категорию
- Применить % скидки
- Экспортировать выбранные
- Удалить
