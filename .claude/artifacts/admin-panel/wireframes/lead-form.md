# Wireframe: Lead Create / Edit Form

URLs:
- `/admin/leads/create`
- `/admin/leads/{id}/edit`

## Подход: Sheet (slide-over) или Full Page

**Рекомендация**: Slide-over (выезжающая панель справа) для быстрого создания, full-page form для глубокого редактирования.

## Валидация

| Поле | Правила |
|------|---------|
| Имя контакта | required, min:2, max:255 |
| Телефон | required, формат `+998 xx xxx-xx-xx`, unique-проверка |
| Email | optional, email-format |
| Источник | required, must be in lead_sources |
| Менеджер | required, must be active user |

## Поведение

- **Inline валидация** — Livewire проверяет при `wire:model.lazy` или blur
- **Дубликаты** — при вводе телефона/email, система ищет похожие лиды и предупреждает
- **Auto-fill компании** — при вводе номера телефона можно подтянуть компанию из истории
- **Save shortcuts** — Ctrl+S = сохранить, Ctrl+Enter = сохранить + создать КП

## Действия после сохранения

1. **"Создать лид"** → редирект на `/admin/leads/{id}` (карточка)
2. **"Сохранить и → КП"** → редирект на `/admin/quotes/create?lead_id={id}` (форма КП с предзаполненным клиентом)
