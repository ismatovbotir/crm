## Компоненты

- **Search** — Livewire-поиск с debounce 300мс
- **Filters** — Alpine для UI, Livewire для запроса
- **View toggle** — список/канбан, состояние в URL (`?view=kanban`)
- **Table** — `<x-table>` с сортировкой колонок (Livewire)
- **Pagination** — серверная (Livewire `->paginate(10)`)
- **Bulk actions** — чекбоксы → массовые действия (назначить менеджера, экспорт, удалить)

## Действия в строке

- `⋯` — открывает dropdown (Просмотр / Изменить / Конвертировать в клиента / Удалить)
- `⊕` — быстрое создание задачи / звонка
- `✎` — inline edit статуса
- `🗑` — удалить (с подтверждением)

## Статусы (badge цвета)

```
New           → primary (синий)
Qualified     → info (голубой)
Contacted     → primary (синий)
In Negotiation → warning (жёлтый)
Won           → success (зелёный)
Lost          → danger (красный)
```

## Адаптив

- На `< md`: фильтры сворачиваются под кнопку "Фильтры"
- На `< sm`: таблица превращается в карточки (одна на строку)
