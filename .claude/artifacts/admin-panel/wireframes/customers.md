## Tabs детали

1. **Обзор** — KPI + реквизиты + контакты + последние сделки/тикеты
2. **Контакты** — все контактные лица с возможностью добавить/изменить
3. **КП** — все коммерческие предложения по этому клиенту
4. **Инвойсы** — все инвойсы + общая статистика оплат
5. **Заказы** — заказы из клиентского кабинета
6. **Тикеты** — вся история обращений
7. **История** — лента всех действий (изменения, документы, действия менеджеров)

## Сегменты (бейджи)

```
VIP        → bg-warning-100 text-warning-700  (золотой)
Active     → bg-success-100 text-success-700  (зелёный)
Inactive   → bg-gray-100 text-gray-700         (серый)
Blocked    → bg-danger-100 text-danger-700    (красный)
New        → bg-primary-100 text-primary-700  (синий)
```

## Quick actions (⋯)

- Создать КП
- Создать инвойс
- Создать тикет
- Конвертировать в портал-аккаунт (создать клиентского пользователя)
- Сменить менеджера
- Архивировать
