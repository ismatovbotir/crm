## Приоритеты (бейджи)

```
Critical  → bg-danger-100 text-danger-700  + 🔴 dot
High      → bg-warning-100 text-warning-700 + 🟠 dot
Normal    → bg-info-100 text-info-700      + 🟡 dot
Low       → bg-gray-100 text-gray-700      + 🟢 dot
```

## SLA-индикатор

- ✓ ok — в рамках SLA
- ⏰ скоро — осталось < 25% времени
- ⚠ просрочено — превышен SLA

## Действия с тикетом

- **Resolve** — закрыть как решённый, попросить CSAT-оценку у клиента
- **Pending** — ждём ответа клиента (SLA-таймер останавливается)
- **Reassign** — передать другому исполнителю
- **Merge** — объединить с другим тикетом (если дубликат)
- **⋯** — Реоткрыть / Изменить категорию / Удалить (admin only)

## Email-уведомления

При каждом сообщении:
- Клиент получает email с текстом + ссылкой на портал
- Менеджер получает push в систему + email при изменении статуса
