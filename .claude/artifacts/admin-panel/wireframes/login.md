## Layout

- Фон: `bg-gray-50` (gradient опционально: `bg-gradient-to-br from-primary-50 to-gray-50`)
- Карточка: центрирована, `max-w-md`, `bg-white rounded-lg shadow-lg p-8`
- Логотип сверху карточки, выровнен по центру
- Поля стандартные `<x-input>`
- Кнопка primary, full-width

## Поведение

- Валидация на клиенте (Alpine.js): обязательные поля
- При ошибке логина: красный alert над полями `<x-alert variant="danger">`
- "Забыли?" → `/forgot-password`
- "Войти в портал" → `/portal/login` (отдельный логин для клиентов)
- После успеха → `/admin` (для внутренних) или `/portal` (для клиентов) — определяется ролью

## Адаптив

- На mobile: карточка занимает всю ширину с `mx-4`
- Логотип размером 48x48 (на mobile может быть 40x40)
